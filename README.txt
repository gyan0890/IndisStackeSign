1.Open this application in visual studio 2010.
2.In solution Explorer you can see "eSign.aspx". Set "eSign.aspx" page as default page.
3.Now Click on run icon.
4.In browser you can see an interfce where you need to put a valid aadhaar number.
5.Click on "Generate OTP" button. After clicking this button you will get an OTP on register mobile number and email id.
6.Now enter otp and give MIME Type as "mime" and choose a file and then click "Submit" button.
7.After "Submit" button click the signed document will apper in pdf format.
